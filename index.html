18天冲刺计划

发表于 2020-04-22 | 分类于 生活

不管做什么，做一做计划，然后尽早开始行动，这是一个好的策略。目前来看，由于调剂的原因，导致最终的目标学校不一样，而且每个学校的复试科目也不相同，像XX大学复试要考查离散数学，而我本科的时候没有学习过这一门课程。在这种情况下，如果没有一个具体的计划的话，很容易学着学着心态会出一些问题，然后陷入效率低下的怪圈。

坚定考研的信念

现在的我，出去找工作根本不占任何优势，没有技术，出身不好，失去了应届生的身份，大学里唯一可以写在简历上面的事情就是水了一篇论文以及打了一个比赛，但是论文与比赛对于找工作来言，没有帮助。具体而言，如果是应届生而且有论文有比赛的话，部分公司会看到你的论文跟比赛会感觉你有潜力，会要你。但是我是往届生，没有工作经验，公司要我之后就要重新开始培养，这是相当不划算的一件事情。

所以说，如果充分挖掘我的论文跟比赛价值的话，还是要选择考研这条路。

目前有这么几种情况

原报上岸

目前来看，线上复试基本确定，复试时间也基本确定，招生老师说大概率取消机试跟笔试。以下讨论建立在取消机试跟笔试的基础上进行讨论。如果是线上复试且取消机试跟笔试的话，那么复试所占的比重大概率会下降。按照群里的排名，我处于复试线外。但是我总感觉群里的成绩排名不是准确的，所以我有一定的概率以倒数的身份进入复试，但是这个概率很低。不过，只要进入复试，就是赚了，因为最起码有一次面试的经历，之后进行调剂复试的话也会有经验一些。

总之，原报上岸的可能性很小。

XXYY实验室

这种情况可能性大一些，但是并不能100%保证可以上岸。

外校调剂

感觉基本上都不靠谱，如果到了这一步，那就是看运气了。

具体计划

目前来看，取消机试跟笔试的话，项目跟基础知识是考查的重点。

论文

每天都要读一遍，然后再进行一些拓展阅读，关键地方要会用英文描述

比赛

没事就去看上次比赛自己录制的音频视频以及PPT，将一些常见的方法整理下来。

口语

听大于说，关键是把老师说的给听清楚。通过B站上面的youtube视频来练习英语听力。自己写几个常见问题的模板，参照小白考研的模板

基础知识

这个比较让人头疼，之前虽然学过数据库，但是好像跟没学过一样。先看面经把常见问题背过。

数据库，计算机网络，操作系统。多看几遍这三门课程的面经。

闲暇时刻

这段时间压力肯定会非常的大，所以怎样放松是一个问题。

看哈佛幸福课

这个课程一共12集，每天看半集，可以撑到复试那一天了。

写博客

目前来看，写博客还是比较解压的

运动

跟着徐灿做运动，每天拿出30分钟来出出汗，特别是不想学习的时候

摸鱼方式总结

发表于 2020-04-22 | 分类于 生活

在家准备复试，没有在学校学习的那种感觉，所以经常对着电脑打发时间，过段时间之后就会感觉到时间都被虚度了，所以想写一篇文章，来总结一下自己摸鱼的方式。

逛论坛

主要逛的论坛有，北邮人论坛，v2ex，豆瓣。目前正在戒除虎扑社区（由于上次徐大SAO事件，让我重新认识了虎扑这个群体）。

刷qq群

主要在qq群里看一下同组同学之间的讨论以及一些调剂的信息。

打游戏

主要打手游，刺激战场，大约打三四十分钟之后就会有眩晕感

刷B站

B站对我的黏性还是很大的，会在上面看一些推荐的视频，不过这些视频我大部分都不感兴趣，热门视频也不感兴趣。会在上面看一些拳击比赛的视频。

看书

其实看书的时间是很少的，大部分时间都花在找书上面了，能够专心读下去的书很少。

浅谈计算机考研与工作

发表于 2020-04-21 | 分类于 生活

我们经常在互联网上看到这一类问题，计算机专业的学生，该考研还是找工作？这是一个小马过河的问题，处在不同状况的人有着不同的选择，这篇文章笔者会讨论一下各种情况并给出自己的建议。

本科双非，技术不好

我是属于这一类人，感觉考研对我来说，就是一个换皮的机会。如果不考研，毕业之后直接找工作的话， 那么找到的工作显然不好，想找大厂工作的话，基本上很难过简历关。辛苦一年，考上理想学校的研究生，在研究生期间不断打磨自己的技术，加上学校的名气的话， 还是有可能获得大厂的offer的。
